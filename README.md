# Приложение для обработки одномерных массивов (WPF)

Программа на языке **C#** с графическим интерфейсом на базе **WPF (Windows Presentation Foundation)**. Приложение позволяет вводить массив чисел, выполнять его валидацию, сортировать элементы методом пузырька и осуществлять поиск заданного значения.

## 📌 Основной функционал

* **Динамический парсинг:** Ввод числовых данных одной строкой. В качестве разделителей поддерживаются пробелы, запятые и точки с запятой.
* **Сортировка:** Классическая пузырьковая сортировка (`Bubble Sort`) массива по возрастанию.
* **Поиск элементов:** Линейный поиск индекса заданного числа в массиве с выводом понятной для пользователя позиции (нумерация с 1).
* **Сброс состояния:** Быстрый возврат интерфейса и внутренних структур данных к исходному виду.
* **Обработка ошибок:** Защита от некорректного ввода (буквы, спецсимволы) с выводом предупреждающих диалоговых окон (`MessageBox`).

## 🛠 Технологический стек

* **Язык программирования:** C#
* **Платформа:** .NET (WPF)
* **Технологии:** LINQ (для парсинга строк в массив данных)
